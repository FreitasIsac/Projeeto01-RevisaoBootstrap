## 📖 Sobre

Landing page moderna e responsiva desenvolvida para um produto SaaS fictício. Este projeto marca a introdução ao uso de Frameworks CSS (Bootstrap 5) para desenvolvimento ágil de interfaces.

 

## 🎯 Conceitos Aprendidos

- **Grid System:** Uso de Rows e Columns para layouts responsivos.

- **Utilitários:** Espaçamento, Cores e Tipografia sem CSS customizado.

- **Componentes:** Navbar Responsiva e Cards.

- **Ícones:** Integração com Bootstrap Icons.

 

## 🛠 Tecnologias

- HTML5

- Bootstrap 5 (CDN)

- CSS3 (Customizações mínimas)
